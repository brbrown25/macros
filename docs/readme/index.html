    Macros

    For latest snapshot use

    libraryDependencies += "com.bbrownsound" % "macros" % "1.0.2-SNAPSHOT"

    And for the latest stable release use

    libraryDependencies += "com.bbrownsound" % "macros" % "1.0.1"

    For ease of useability, this project is cross compiled against 2.11.12, 2.12.12, 2.13.0, 2.13.1, 2.13.2, 2.13.3, 2.13.4
